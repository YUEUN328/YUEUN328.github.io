## TIL

* id, name 이름은  모델 클래스와 동일하게 쓰자. 그럼 고민 안 해도 돼서 편하다.

<br>

* 과제

  1. users 테이블 전체 검색 

     select * from users;   

  2. users 테이블 신규 회원 등록

     insert into users (...) values (...);

  3. users 테이블 상세 보기

     select * from users where userid = ?
